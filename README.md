# Resume Tool

Un éditeur de CV HTML simple et puissant, sans backend ni dépendances.

## Fonctionnalités

- **Édition directe** - Cliquez sur n'importe quel texte pour le modifier
- **Sauvegarde automatique** - Vos modifications sont sauvegardées dans le navigateur
- **Profils multiples** - Créez différentes versions de votre CV (tech, management, etc.)
- **Export/Import JSON** - Sauvegardez et restaurez vos profils
- **Personnalisation complète** :
  - 6 couleurs personnalisables (primaire, accent, titres, dates, icônes, bordures)
  - Taille des barres de compétences
  - Niveau individuel de chaque compétence
  - Zoom du contenu principal
- **Sélecteur de photo** - Chargez votre photo de profil
- **Switch FR/EN** - Changez les labels de section
- **Impression A4** - Export PDF propre via le navigateur

## Utilisation

1. Ouvrez `resume.html` dans votre navigateur
2. Modifiez le contenu en cliquant sur le texte
3. Personnalisez les couleurs et styles via la barre d'outils
4. Sauvegardez des profils pour différentes candidatures
5. Imprimez ou exportez en PDF (Ctrl+P)

## Stack technique

- HTML5
- CSS3 (Grid, Variables, Print media queries)
- JavaScript vanilla
- localStorage pour la persistance
- Font Awesome (CDN)
- Google Fonts - Poppins (CDN)

**Zéro dépendance locale, zéro build, zéro backend.**
